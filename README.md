# Log Analyzer Tool


### Overview

This tool was created to quickly analyze and find suspicious activite from a given log.txt file for quick and effective log analysis.
It works by seperating various data from the sample_log.txt such as AUTH_SUCCESS and PRIV_CHANGE and will store the data to be pulled for later use in the summary.txt

### How to use this tool:
First you will need to have a file provided [title]here!(https://github.com/CHarris-VR/code-you-capstone/blob/main/sample_log.txt) or by clicking on the link above to the sample_log.txt
